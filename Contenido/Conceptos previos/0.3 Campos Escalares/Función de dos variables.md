Sea D el conjunto de pares ordenados de números reales. Si a cada par ordenado *(x, y)* de D le corresponde un único numero real *f(x, y)*, entonces se dice q f es una función de x e y. El conjunto D es el dominio de *f*, y el correspondiente conjunto de valores *f(x, y)* es el rango de *f*
